# Devboard
a devboard i hacked together with minimal sleep
total jlcpcb cost $38.9 (including local taxes and shipping)
5pcs pcb and 2pcs pcba

# BOM is the Devboard.csv file

# Summary:
Raspberry rp2040 microcontroller 
gpio 0-25 broken out, gpio26-29/adc0-3 also broken out 

similar pinout as raspberry pi pico, but with the gpio 25 broken out, trading a ground pin for a gpio

power pins moved to the top right of the board in a group, both vbus and 3v3
